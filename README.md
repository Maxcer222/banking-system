# 🏦 Banking Management System

A professional-grade console application written in **C++** that simulates the core operations of a banking institution. This project focuses on **Object-Oriented Programming (OOP)**, memory management, and robust financial logic.

---

## ✨ Key Features
* **Account Operations:** Create new accounts, update information, and manage customer records.
* **Transaction System:** Support for deposits and withdrawals with real-time balance calculations.
* **Data Security:** Implementation of basic logic to ensure valid transactions and prevent overdrafts.
* **Search & Inquiry:** Quickly find account details and view current status via the console.
* **Clean UI:** User-friendly menu-driven interface for easy navigation.

---

## 🛠️ Tech Stack
* **Language:** C++ (Modern standards)
* **Paradigm:** Object-Oriented Programming (Classes, Encapsulation)
* **Development Tools:** Developed in MS Visual Studio / Compiled with MSVC

---

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone [https://github.com/Maxcer222/banking-system.git](https://github.com/Maxcer222/banking-system.git)
cd banking-system
```


### 2. Compilation
### Using G++:

```bash
g++ ConsoleApplication1.cpp -o BankingSystem
```

### Using Visual Studio:

1. Open the folder in Visual Studio.

2. Build the project (Ctrl+Shift+B).

3. Run the application (F5).

📦 Project Structure
- ConsoleApplication1.cpp — Main source code containing the logic.
- ConsoleApplication1.vcxproj — Project configuration for Visual Studio.
- /x64/Debug — Compiled binaries and temporary build files.
